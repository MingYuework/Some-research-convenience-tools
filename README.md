## 📖 工具详情

### 1. 🧪 Bader 电荷分析工具

> Bader Charge Analysis Tool

适用于 **VASP + Bader** 计算流程的 ACF.dat 本地分析工具。帮助研究人员快速计算电荷在分子片段间的转移量。

**功能 / Features:**
- 📋 直接粘贴 `ACF.dat` 文本内容，解析所有原子的 Bader 电荷
- 🧩 多片段定义：按原子序号范围划分分子片段，支持任意数量片段
- ⚡ 一键计算各片段在复合体系中的得失电子数
- 🎨 结果自动着色：得电子（绿色）、失电子（红色）
- 🔢 支持自定义各片段纯相（复合前）的总价电子数基准

**使用场景:** 表面吸附、分子间电荷转移、掺杂体系电荷分析等第一性原理计算研究。

<details>
<summary>📸 界面预览 (Screenshot)</summary>

**操作流程:**
1. 用记事本打开 `ACF.dat`，全选复制，粘贴到文本框 → 点击「解析数据」
2. 设置片段：输入各片段的原子序号范围（对应 POSCAR 顺序）和纯相总电子数基准
3. 点击「一键计算界面得失」→ 查看转移结论

</details>

### 2. 🖼️ 论文组图工具

> Paper Figure Layout Tool by 明月

面向学术论文投稿的图片排版工具。支持拖拽式操作，灵活控制网格布局，高 DPI 导出，一站式完成组图工作。**最新版本 V2.0，推荐使用。**

| 版本 | 文件 |
|------|------|
| **V2.0** (最新) | `PPT组图工具/组图工具V2.0.html` |
| V1.0 | `PPT组图工具/组图工具V1.0.html` |

**功能 / Features:**

**布局模式:**
- 🔲 **均匀网格:** 标准的 N×M 网格排列（1×2 ~ 4×4 预设 + 自定义）
- ⭐ **奇数图片:** N×M 网格 + 1 张独立大图，支持上/下/左/右放置
- 🎛️ **自定义:** 每行独立设置图片数量，灵活控制列宽分配

**图片管理:**
- 📂 内置资源管理器：拖放文件/文件夹批量导入图片
- 🔍 图片预览大图模式：支持缩放、键盘左右切换
- ✏️ 双击文件重命名
- 🖱️ 拖拽图片到网格单元格，也可从资源管理器拖入

**编辑功能:**
- ✂️ 图片裁剪：支持自由比例、固定比例（1:1/4:3/16:9）、匹配单元格
- 🔍 每格独立缩放和平移（鼠标滚轮缩放 + 拖拽平移）
- 🏷️ 标签系统：(a)(b)(c)… 支持方向键和按钮微调位置
- 🎨 自定义标签字体、字号、颜色

**导出:**
- 📥 支持 PNG / JPEG 格式
- 📐 可调 DPI：96 / 150 / 200 / **300 (推荐)** / 400 / 600
- 🖊️ 可选导出黑色单元格边框
- ⌨️ 快捷键：`Ctrl+E` 快速导出

#### 🆕 V2.0 新增功能

| 新功能 | 说明 |
|--------|------|
| 🏷️ **多格式标签** | 8 种编号格式：(a)(b)(c) / (A)(B)(C) / (1)(2)(3) / a b c / A B C / 1 2 3 / (i)(ii)(iii) / (I)(II)(III) |
| ✏️ **自定义标签** | 双击标签直接编辑为任意文字（如 "Fig.1", "实验组"） |
| 📍 **标签位置** | 支持框内 / 框外左上角两种放置方式 |
| ↔️ **独立间距** | 水平间距和垂直间距可分别设置 |
| 📏 **独立边距** | 水平外边距和垂直外边距可分别设置 |
| 🔎 **页面缩放** | 工具栏增加全局缩放按钮（+ / − / 1:1），配合 `Ctrl+滚轮` |
| 📐 **宽高独立缩放** | 每个单元格分别显示宽/高缩放百分比 |
| 🎛️ **侧栏重构** | UI 更紧凑，操作更高效 |

**使用场景:** SCI 论文投稿组图、学位论文图片排版、学术海报图片整理。

<details>
<summary>📸 操作流程</summary>

1. 拖放图片文件/文件夹到左侧资源管理器
2. 选择布局模式（均匀网格 / 奇数图片 / 自定义）
3. 调整网格参数（行列数、单元格大小、独立水平/垂直间距）
4. 从资源管理器拖入图片到单元格
5. （可选）裁剪图片、调整显示区域、移动/编辑标签
6. 选择 DPI 和格式 → 导出图片

</details>

### 3. 🎨 简易三维绘制工具 (V1.0)

> Simple 3D Model Drawing Tool

基于 Three.js 的浏览器端三维几何建模工具。支持基础几何体创建、布尔运算和多种格式导出。

**功能 / Features:**
- 🔷 **基础几何体:** 长方体、圆柱体、球体、圆锥体、圆环 — 参数可调
- 🔄 **旋转控制:** 绕自身中心旋转 / 绕参考点公转
- 📋 **复制平移:** 复制模型到指定位置、XYZ 三轴平移
- 🧮 **布尔运算 (CSG):** 差集 (A-B)、并集 (A+B)、交集 (A∩B)
- 📦 **格式导出:** STL 格式、OBJ 格式
- 🖱️ **3D 操控:** 轨道旋转、缩放、平移视角
- 💡 带网格地面、坐标轴辅助线、半球光照 + 环境光

**依赖:** 运行时通过 CDN 加载 Three.js (v0.160.0) 和 three-csg-ts，无需本地安装。

**使用场景:** 科研模型示意图、TOC 图形绘制、3D 打印模型快速原型、教学演示。

<details>
<summary>📸 界面预览 (Screenshot)</summary>

**操作流程:**
1. 选择几何体类型 → 设置参数 → 点击「生成模型」
2. 选中模型后可旋转、复制、平移
3. 选中 2 个模型后可进行布尔运算
4. 导出为 STL 或 OBJ 格式

</details>

## ⚠️ 注意事项

1. **浏览器兼容性:** 推荐使用 Chrome / Edge / Firefox 最新版本。三维绘制工具需要 WebGL 支持。
2. **大文件处理:** 组图工具导出高 DPI 图片时可能消耗较多内存，建议分批处理超大图片。
3. **数据隐私:** 所有工具均为纯客户端运行，图片和文件不会离开你的电脑。
4. **三维绘制依赖:** 需要联网加载 Three.js CDN 资源（首次加载后浏览器会缓存）。
